# 🏦 Banco Java — Sistema Bancário via Console

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=java&logoColor=white)
![Console App](https://img.shields.io/badge/Console-App-green)
![Nível](https://img.shields.io/badge/Nível-Iniciante-yellow)

Aplicação de console em Java que simula operações básicas de uma conta bancária — consultar saldo, receber depósitos e realizar transferências — com menu interativo e loop de controle.

---

## Funcionalidades

- Consultar saldo atual da conta
- Receber valor (depósito)
- Transferir valor (débito)
- Menu em loop com opção de saída

---

## Como executar

```bash
# Compile
javac Main.java

# Execute
java Main
```
